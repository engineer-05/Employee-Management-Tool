# 职工管理系统 (Employee Management System)

一个基于 C++ 面向对象编程的**控制台职工信息管理系统**，支持对职工信息的增删改查、排序和持久化存储。

---

## 📋 项目简介

本系统采用 C++ 多态特性设计，通过抽象基类 `Worker` 派生出三种职工角色：**普通员工（Employee）**、**经理（Manager）** 和 **总裁（Boss）**。系统通过控制台菜单与用户交互，数据以文本文件 `empfile.txt` 形式持久化存储，每次启动自动加载已有数据。

---

## 🏗️ 项目结构

```
├── main.cpp            # 程序入口，主菜单循环
├── worker.h            # 职工抽象基类声明
├── employee.h          # 普通员工类声明
├── employee.cpp        # 普通员工类实现
├── menager.h           # 经理类声明
├── menager.cpp         # 经理类实现
├── boss.h              # 总裁类声明
├── boss.cpp            # 总裁类实现
├── workerManager.h     # 管理类声明（核心业务逻辑）
├── workerManager.cpp   # 管理类实现
├── empfile.txt         # 职工数据文件
└── main                # 编译后的可执行文件
```

---

## 🚀 功能特性

| 功能编号 | 功能描述 |
|:---:|:---|
| **0** | 退出系统 |
| **1** | **增加职工** — 批量添加新职工，选择职位（员工/经理/总裁） |
| **2** | **显示职工** — 以列表形式展示所有职工详细信息 |
| **3** | **删除职工** — 按编号删除指定职工（离职处理） |
| **4** | **修改职工** — 按编号修改职工的全部信息（编号、姓名、职位） |
| **5** | **查找职工** — 支持按编号、姓名、职位三种方式查找 |
| **6** | **排序职工** — 支持按编号升序 / 按职位编号降序排列 |
| **7** | **清空数据** — 一键清空所有职工记录（带确认提示） |

## 🧱 类设计

```
Worker（抽象基类）
├── id         : int        职工编号
├── name       : string     职工姓名
├── deptId     : int        职位编号 (1=员工, 2=经理, 3=总裁)
├── showInfo()             纯虚函数 — 显示个人信息
└── getDeptName()          纯虚函数 — 返回职位名称

    ┌──────────┼──────────┐
    ▼          ▼          ▼
Employee    Menager     Boss
 (员工)      (经理)      (总裁)

workerManager（管理类）
├── EmpArray  : Worker**   职工指针数组
├── EmpNum    : int        职工总数
├── FileIsEmpty : bool     文件是否为空
├── Add_Emp()              添加职工
├── show_Emp()             显示所有职工
├── delete_Emp()           删除职工
├── mod_Emp()              修改职工
├── find_Emp()             查找职工
├── sort_Emp()             排序职工
├── save()                 保存到文件
├── init_Emp()             从文件初始化
└── Clean_File()           清空数据
```

## 🔧 编译与运行

### 环境要求

- 支持 C++11 或以上标准的编译器（g++ / MSVC / clang++）

### 编译

```bash
g++ -o main main.cpp employee.cpp menager.cpp boss.cpp workerManager.cpp
```

### 运行

```bash
./main
```

运行后将看到系统菜单，输入对应数字即可操作。

## 📊 数据存储格式

职工数据存储在 `empfile.txt` 中，每行一条记录，格式为：

```
<编号> <姓名> <职位编号>
```

**示例数据：**

```
32 孙悟空 3
8 王五 2
53 沙和尚 2
56 赖六 2
5 赖六 1
43 猪八戒 1
90 李扒皮 1
```

其中职位编号含义：`1` = 普通员工, `2` = 经理, `3` = 总裁。

## 🎯 技术要点

- **多态** — 通过 `Worker` 抽象基类统一接口，利用虚函数实现运行时多态
- **动态内存管理** — 使用 `Worker**` 二级指针管理职工对象数组，支持动态扩容
- **文件流操作** — 使用 `ifstream` / `ofstream` 实现数据的读写持久化
- **选择排序** — 排序功能使用选择排序算法，按编号或职位编号排序后自动回写文件
